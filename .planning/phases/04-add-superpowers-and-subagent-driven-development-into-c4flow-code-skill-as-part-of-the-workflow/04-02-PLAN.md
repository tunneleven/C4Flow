</objective>

   - Assert `references/workflow-state.md` contains `skills/code/SKILL.md`.
   - Assert the CODE row still exists.

   - Assert `references/phase-transitions.md` mentions `beads or \`tasks.md\``.

   - Assert `skills/code/SKILL.md` documents missing-prerequisite recovery commands.
